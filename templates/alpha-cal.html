DB Calendaring Project

JavaScript was used to speed up data entry and calendaring in an existing database front end (Alpha Five / SQL Server).

Chess Moves Project
The Start Date is taken from a legally binding agreement to settle an intellectual property dispute including payment for release of liability. Optional provisions can include making installment payments, required purchase of products within a certain time frame, and a certification of continued compliance. Reminders are scheduled for each provision; they are interdependent and cannot be calculated from the Start Date alone. Alpha Five sends field values to the SQL database.

Results


= Start Date + Installments + Purchase Period JS prompt

= Start Date + Installments + 3 months if Purchase Period is between 6 and 24 months


= Start Date + Installments + 6 months if Purchase Period is at least 24 months


= Certificate Reminder 1 Date if they are within 45 days of each other

= Purchase Reminder 1 Date + 6 months


= Certificate Reminder 1 Date if they are within 45 days of each other

= Purchase Reminder 1 Date + 12 months
Each of the fields above required manual calculation and typing. The instructions had to be memorized or found in a separate location as the notes on this page aren't on the real page. The JavaScript addition saved time, reduced errors, and allowed new users to easily learn the system.
